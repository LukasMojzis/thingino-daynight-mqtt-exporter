# Thingino daynight MQTT exporter
Sends daynight status to MQTT.
Requires valid `send2.json` config.

## install
`curl -sSL https://github.com/LukasMojzis/thingino-daynight-mqtt-exporter/raw/refs/heads/master/install.sh | sh`

## uninstall
`curl -sSL https://github.com/LukasMojzis/thingino-daynight-mqtt-exporter/raw/refs/heads/master/uninstall.sh | sh`
